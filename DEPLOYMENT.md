# Deployment Guide

To get a public web link (e.g., `https://your-project.vercel.app`), you can deploy this static site to Vercel.

## Option 1: Vercel CLI (Fastest)

1.  **Install Vercel CLI** (if not installed):
    ```bash
    npm install -g vercel
    ```

2.  **Deploy**:
    Open your terminal in this directory (`C:\Users\91891\.gemini\antigravity\scratch\food-rescue-static`) and run:
    ```bash
    vercel
    ```

3.  Follow the prompts (Hit Enter to accept defaults).
4.  Vercel will give you a **Production** URL.

## Option 2: Drag & Drop (No Code)

1.  Go to [vercel.com/new](https://vercel.com/new).
2.  Drag and drop the `food-rescue-static` folder into the window.
3.  Click **Deploy**.
4.  Vercel will generate your web link instantly.
